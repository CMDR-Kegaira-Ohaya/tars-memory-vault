# 01_SYSTEM_MAP.md

## Repo planes

### System plane
`repo-manual/`
Canonical repo-side guidance, routing, references, procedures, and troubleshooting.

### Work plane
`work/`
Active drafting and project work by operator or shared context.

### Content plane
`collections/`
Domain/content holdings such as entertainment, books, and various.

### History plane
`logs/`
Decision records and incident records.

### Automation plane
`.github/workflows/`
Executable GitHub workflow files.
