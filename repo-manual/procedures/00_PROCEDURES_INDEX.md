## Purpose
This file is the entry point for the repo-side procedure layer.

## Scope
Procedures are split by operational intent:
- `build/` – how to construct or install something in the repo
- `runtime/` – how to operate or use a system after it exists
- `maintenance/` – how to keep, tighten, repair, or revise something over time
