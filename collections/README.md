# COLLECTIONS.md

## Purpose
This file is the landing surface for the repo content/holdings layer.

## Scope
- `entertainment/` – film, series, games, music, media or other entertainment-oriented holdings
- `books/` – books, authors, notes, or book-related material
- `various/` – miscellaneous or not-yet-split material that does not clearly belong elsewhere

## Working rule
Use `collections/` for subject matter and holdings.
Do not use it for repo-guidance, operating procedures, or decision logs.

## Current state
This area is still mostly empty by design.
The next step is to place real material in the correct subdirectory only when the content is ready.
