# Import Bay Sample Entry

This sample package exists to validate the authenticated collections-ingest path.

It is a canonical repo entry under `/collections/` with a package root, package manifest, and content file.

Use it to verify that the flow now distinguishes:

- local terminal staging
- authenticated repo ingest
- repo-indexed catalogue visibility
- operator-facing request history and repo-verified response sidecars

The content here is intentionally small and plain so the first before/after test can focus on routing, not presentation.
