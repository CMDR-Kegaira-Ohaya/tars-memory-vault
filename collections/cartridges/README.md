## Purpose
This directory is the canonical home for terminal-runtime-compatible cartridges.

## Working rule
A cartridge stored here is a catalogue item under `collections/`, but it is also eligible for terminal mounting once a compatible runtime/manifest exists.

## Scope
Use this directory for:
- text adventures
- tiny roguelikes
- puzzle cartridges
- board/card microgames
- other small fixed-scope interactive artifacts intended for the terminal

## Not the same as other collection families
Items in `collections/books/` or other collection families may still be browseable in the broader catalogue, but they are not mountable cartridges unless they gain a cartridge adapter or runtime shape later.

## Current state
This directory is established now so future cartridge work has a stable home before real cartridges arrive.
