# Import Bay Sample Entry

This repo-backed sample mirrors the shape used for import compatibility tests.

## What it proves

- repo files can be selected from `Load -> Repo Load`
- a selected file can become the active Home item
- Home can render readable text instead of only state labels

## Note

This is still a simple markdown reader example.
The import pipeline and the runtime pipeline remain separate behind the scenes, even when the frontend keeps the interaction simple.
