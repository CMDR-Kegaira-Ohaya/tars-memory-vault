# Example Cartridge

This is the first readable sample mounted into **Home**.

The important behavior is simple:
- select from a browser surface
- load into Home
- read the active item here
- use `Eject` to clear it

## Why this file exists

Previously, the terminal could say an item was loaded while Home still showed no actual readable content. This sample gives the mounted reader something real to display.

## Sample text

TARS terminal should separate selection from experience.
Home is the experience surface.
Cartridges, Boards, and Load are chooser surfaces.

> If Home says a file is loaded, the file should be visible there.
