# Procedure Index

This file maps operational procedures to their manual locations.

TARS consults this file before scanning the manual.

Format:

PROCEDURE_NAME → relative/manual/path

Example:

EXPORT_PREFLIGHT → appendices/APP_G_EXPORT_PREFLIGHT.md
REPO_IMPORT → appendices/APP_H_REPO_IMPORT.md
RELAY_CHECK → appendices/APP_I_RELAY_CHECK.md
UPLOAD_PACKAGE → appendices/APP_J_UPLOAD_PACKAGE.md
