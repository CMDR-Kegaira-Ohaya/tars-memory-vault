# toolkit (T1RS)
This is TARS' toolkit (tool-cabinet). Content here is curated and reuseable, but it is **not binding** by default.

- **Toolkit ≠ Binding**: adding a file here does not change TAR behavior.
- Activation requires explicit Governance + scope (and mutual approval for shared defaults).
