# /ops — CONTRIBUTING

- /vault = binding truths (rare; explicit Governance + scope)
- /toolkit = reusable tools (inert unless invoked)
- /ops = maintenance + evolution handbook

If unsure: default to /toolkit or /ops (not /vault).

Shared changes:
- state scope (Personal A / Personal B / Shared)
- if other operator absent and scope is Shared: defer (inactive proposal)

If you edit /ops, log it in /ops/CHANGELOG.md (date, what, why, scope).

Notes:
- The curated reading library lives at /toolkit/library/.
